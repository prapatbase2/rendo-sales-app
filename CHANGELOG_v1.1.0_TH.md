# Rendo PWA v1.1.0 — รายการแก้ไข

- แก้หน้ารายเดือนและหน้าลงวันทำงานค้าง
- เพิ่ม timeout, cache fallback และป้องกันโหลดซ้อน
- ปรับการบันทึกวันทำงานให้คืนสถานะปุ่มเสมอ
- ลดการชนกันของ Auto Backup และป้องกันสำรองซ้อน
- ปรับหน้าค่าตอบแทนเป็นทีละคน พร้อมแถบเลื่อนชื่อ
- เพิ่มช่องเพดานฐานเงินเดือนประกันสังคมสำหรับเจ้าของ
- เพิ่มรายละเอียดรายงานค่าตอบแทนครบจนถึงยอดโอนปลายเดือน
- แก้ PDF หน้าขาว และเพิ่มตรวจสอบไฟล์ก่อนดาวน์โหลด
- เพิ่มแชร์ PDF ผ่านหน้าต่างแชร์ของมือถือเพื่อส่งเข้า LINE
- เพิ่ม fallback ดาวน์โหลด PDF + คัดลอกข้อความเมื่ออุปกรณ์ไม่รองรับการแชร์ไฟล์
- อัปเดต Service Worker เป็น build 1.1.0 เพื่อรับไฟล์ใหม่ได้ง่ายขึ้น
